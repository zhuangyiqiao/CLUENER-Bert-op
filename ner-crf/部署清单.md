

## ⚠️ 注意事项

1. **首次运行会下载预训练模型** (~400MB)
   - 确保有足够网络空间
   - 模型会缓存在 `~/.cache/huggingface/hub/`

2. **GPU 与 CPU**
   - 代码会自动检测 GPU (`cuda`)
   - 如果无 GPU，会自动使用 CPU（较慢）

3. **显存要求**
   - Batch Size 32：需要 ~8GB 显存
   - 可在 `configs/bert_crf.json` 中调整 batch size

4. **模型权重**
   - 最佳模型会保存为 `outputs/best.pt`
   - 可用于后续预测或微调

## 📌 常见问题

**Q: 如果训练中断怎么办？**  
A: 重新运行 `python scripts/train.py` 会覆盖之前的训练。建议手动保存中间的 checkpoint。

**Q: 如何只验证不训练？**  
A: 运行 `python verify_setup.py` 和 `python test_data_loading.py` 验证环境。

**Q: 如何调整超参数？**  
A: 修改 `configs/bert_crf.json` 中的对应参数，然后重新运行训练。

**Q: 预测脚本什么时候用？**  
A: 训练完成后，使用 `python scripts/predict.py --model_path outputs/best.pt` 进行预测。
